Variables - Storing Information

Variables are like labeled boxes that store data. You can put things in, take things out, and change what's inside.

Creating Variables

Use let or const to create a variable:

let name = "Sarah"; // Can be changed later const age = 15; // Cannot be changed
üí° let vs const:
‚Ä¢ Use const when the value won't change (like your birthday)
‚Ä¢ Use let when the value might change (like your age)

Data Types

Variables can hold different types of data:

1 String - Text (always in quotes)
let firstName = "Alice"; let greeting = "Hello, world!"; let emoji = "üéâ";
2 Number - Numbers (no quotes)
let age = 15; let price = 19.99; let temperature = -5;
3 Boolean - True or False
let isStudent = true; let hasLicense = false; let isRaining = true;
4 Array - List of items
let colors = ["red", "blue", "green"]; let numbers = [1, 2, 3, 4, 5]; let mixed = ["hello", 42, true];

Using Variables

let name = "Alex"; console.log(name); // Shows: Alex let score = 10; score = score + 5; // score is now 15 console.log(score); // Shows: 15

Math with Variables

let x = 10; let y = 5; console.log(x + y); // 15 (addition) console.log(x - y); // 5 (subtraction) console.log(x * y); // 50 (multiplication) console.log(x / y); // 2 (division) console.log(x % y); // 0 (remainder/modulo)

Combining Strings

let firstName = "Sarah"; let lastName = "Smith"; // Method 1: Using + let fullName = firstName + " " + lastName; console.log(fullName); // "Sarah Smith" // Method 2: Template literals (modern way) let greeting = `Hello, ${firstName}!`; console.log(greeting); // "Hello, Sarah!"

Accessing Array Items

let fruits = ["apple", "banana", "orange"]; console.log(fruits[0]); // "apple" (first item) console.log(fruits[1]); // "banana" (second item) console.log(fruits[2]); // "orange" (third item) // Arrays start counting at 0!

Functions - Reusable Code Blocks

Functions are like recipes - you write them once, then use them whenever you need them!

Creating a Function

function sayHello() { console.log("Hello!"); } // Use (call) the function sayHello(); // Shows: Hello!

Functions with Parameters

Parameters let you pass information into a function:

function greet(name) { console.log("Hello, " + name + "!"); } greet("Sarah"); // Shows: Hello, Sarah! greet("Alex"); // Shows: Hello, Alex! greet("Emma"); // Shows: Hello, Emma!

Multiple Parameters

function add(a, b) { let result = a + b; console.log(result); } add(5, 3); // Shows: 8 add(10, 20); // Shows: 30

Returning Values

Functions can send back (return) a value:

function multiply(x, y) { return x * y; } let result = multiply(4, 5); console.log(result); // Shows: 20 // You can use the result directly let total = multiply(3, 7) + 10; console.log(total); // Shows: 31 (21 + 10)

Real-World Examples

Calculate Area of Rectangle
function calculateArea(width, height) { let area = width * height; return area; } let roomArea = calculateArea(10, 12); console.log("Room area: " + roomArea + " sq ft"); // Shows: Room area: 120 sq ft
Convert Temperature
function celsiusToFahrenheit(celsius) { let fahrenheit = (celsius * 9/5) + 32; return fahrenheit; } let temp = celsiusToFahrenheit(25); console.log(temp); // Shows: 77
Check if Even or Odd
function isEven(number) { if (number % 2 === 0) { return true; } else { return false; } } console.log(isEven(4)); // true console.log(isEven(7)); // false

Arrow Functions (Modern Syntax)

A shorter way to write functions:

// Traditional function function double(x) { return x * 2; } // Arrow function (same thing!) const double = (x) => { return x * 2; } // Even shorter (when returning one thing) const double = x => x * 2;
üí° Why Functions Are Awesome:
‚Ä¢ Write code once, use it many times
‚Ä¢ Break big problems into smaller pieces
‚Ä¢ Make code easier to read and understand
‚Ä¢ Fix bugs in one place instead of everywhere
Common Mistakes:
‚ùå Forgetting parentheses: sayHello;
‚úÖ Call with parentheses: sayHello();

‚ùå Wrong order: Call before defining
‚úÖ Define function first, then call it

Conditionals - Making Decisions

Conditionals let your code make decisions based on conditions. Think "IF this is true, THEN do that."

The if Statement

let age = 16; if (age >= 18) { console.log("You can vote!"); }

if...else

Do one thing if true, another thing if false:

let temperature = 25; if (temperature > 30) { console.log("It's hot!"); } else { console.log("It's nice!"); }

if...else if...else

Check multiple conditions:

let score = 85; if (score >= 90) { console.log("Grade: A"); } else if (score >= 80) { console.log("Grade: B"); } else if (score >= 70) { console.log("Grade: C"); } else { console.log("Grade: F"); } // Shows: Grade: B

Comparison Operators

Used to compare values:

// Equality 5 === 5 // true (equal to) 5 !== 3 // true (not equal to) // Comparison 10 > 5 // true (greater than) 10 < 5 // false (less than) 10 >= 10 // true (greater than or equal) 5 <= 10 // true (less than or equal)
üí° Important: Use === (three equals) for comparison, not = (one equals)!
‚Ä¢ = assigns a value: x = 5
‚Ä¢ === compares values: x === 5

Logical Operators

Combine multiple conditions:

let age = 20; let hasLicense = true; // AND - both must be true if (age >= 16 && hasLicense) { console.log("You can drive!"); } // OR - at least one must be true if (age < 13 || age > 65) { console.log("Discount available!"); } // NOT - reverses the condition if (!hasLicense) { console.log("You need a license"); }

Real-World Examples

Password Validator
function checkPassword(password) { if (password.length < 8) { console.log("Password too short!"); } else if (password.length > 20) { console.log("Password too long!"); } else { console.log("Password is good!"); } } checkPassword("abc"); // Password too short! checkPassword("mySecure123"); // Password is good!
Age Category
function getAgeGroup(age) { if (age < 13) { return "Child"; } else if (age < 20) { return "Teenager"; } else if (age < 65) { return "Adult"; } else { return "Senior"; } } console.log(getAgeGroup(15)); // Teenager
Weekend Checker
function isWeekend(day) { if (day === "Saturday" || day === "Sunday") { return true; } else { return false; } } console.log(isWeekend("Saturday")); // true console.log(isWeekend("Monday")); // false

Ternary Operator (Shortcut)

A shorter way to write simple if...else:

// Long way let age = 20; let status; if (age >= 18) { status = "adult"; } else { status = "minor"; } // Short way (ternary) let status = age >= 18 ? "adult" : "minor";
Common Mistakes:
‚ùå if (age = 18) (assignment, not comparison)
‚úÖ if (age === 18)

‚ùå if age > 18 (missing parentheses)
‚úÖ if (age > 18)

Loops - Repeating Actions

Loops let you repeat code multiple times without writing it over and over. Perfect for processing lists or doing something many times!

The for Loop

Use when you know how many times to repeat:

for (let i = 0; i < 5; i++) { console.log("Count: " + i); } // Shows: Count: 0, Count: 1, Count: 2, Count: 3, Count: 4

Breaking it down:

  • let i = 0 - Start at 0
  • i < 5 - Keep going while i is less than 5
  • i++ - Add 1 to i each time (same as i = i + 1)

Looping Through Arrays

let fruits = ["apple", "banana", "orange"]; for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); } // Shows: apple, banana, orange

The while Loop

Use when you don't know how many times to repeat:

let count = 0; while (count < 3) { console.log("Count is: " + count); count++; } // Shows: Count is: 0, Count is: 1, Count is: 2
‚ö†Ô∏è Warning: Be careful with while loops! If the condition never becomes false, you'll create an infinite loop that crashes your browser!

Modern Array Loops

1 forEach - Loop through each item
let colors = ["red", "green", "blue"]; colors.forEach(function(color) { console.log(color); }); // Or with arrow function: colors.forEach(color => console.log(color));
2 for...of - Cleaner array loop
let numbers = [1, 2, 3, 4, 5]; for (let num of numbers) { console.log(num * 2); } // Shows: 2, 4, 6, 8, 10

Real-World Examples

Sum All Numbers in Array
function sumArray(numbers) { let total = 0; for (let i = 0; i < numbers.length; i++) { total = total + numbers[i]; } return total; } let scores = [85, 90, 78, 92]; console.log(sumArray(scores)); // Shows: 345
Find Largest Number
function findMax(numbers) { let max = numbers[0]; // Start with first number for (let num of numbers) { if (num > max) { max = num; } } return max; } let temps = [72, 85, 91, 68, 77]; console.log(findMax(temps)); // Shows: 91
Count Down
function countDown(start) { for (let i = start; i >= 0; i--) { console.log(i); } console.log("Blast off!"); } countDown(5); // Shows: 5, 4, 3, 2, 1, 0, Blast off!
Build HTML List
let items = ["Milk", "Eggs", "Bread"]; let html = ""; for (let item of items) { html += `<li>${item}</li>`; } console.log(html); // Shows: <li>Milk</li><li>Eggs</li><li>Bread</li>

Loop Control

// break - stop the loop early for (let i = 0; i < 10; i++) { if (i === 5) { break; // Stop when i is 5 } console.log(i); } // Shows: 0, 1, 2, 3, 4 // continue - skip to next iteration for (let i = 0; i < 5; i++) { if (i === 2) { continue; // Skip when i is 2 } console.log(i); } // Shows: 0, 1, 3, 4 (skips 2)
Common Mistakes:
‚ùå for (let i = 0; i < 5) (missing i++)
‚úÖ for (let i = 0; i < 5; i++)

‚ùå Infinite loop: while (true) { ... }
‚úÖ Always have a condition that becomes false!

Debugging - Finding and Fixing Errors

Debugging is the process of finding and fixing errors in your code. Every programmer spends a LOT of time debugging - it's a normal part of coding!

Types of Errors

1 Syntax Errors - Typos in your code
console.log("Hello" // Missing closing )
let name = Sarah; // Missing quotes
if (x > 5 { // Missing )

These prevent your code from running at all. Check the console for error messages!

2 Logic Errors - Code runs but does wrong thing
// Bug: Should add, not subtract function add(a, b) { return a - b; // Oops! Wrong operator }

Code runs but gives wrong results. Use console.log() to find these!

3 Runtime Errors - Errors that happen while running
let numbers = [1, 2, 3]; console.log(numbers[10]); // undefined - index doesn't exist

Using console.log()

Your #1 debugging tool! Use it to see what's happening in your code:

function calculateTotal(price, quantity) { console.log("Price:", price); // Check the inputs console.log("Quantity:", quantity); let total = price * quantity; console.log("Total:", total); // Check the result return total; }

Debugging Strategies

1. Print Everything
Add console.log() statements to see values at each step:
function average(numbers) { console.log("Input array:", numbers); let sum = 0; for (let num of numbers) { sum += num; console.log("Current sum:", sum); } let avg = sum / numbers.length; console.log("Final average:", avg); return avg; }
2. Comment Out Code
Use // to temporarily disable code and find where the error is:
function processData() { getData(); // This works // validateData(); // Comment this out to test // saveData(); // Comment this out to test }
3. Check the Console
Press F12 and click "Console" to see:
‚Ä¢ Error messages (in red)
‚Ä¢ Your console.log() output
‚Ä¢ Line numbers where errors occur
4. Read Error Messages Carefully
Error messages tell you exactly what's wrong:
Uncaught ReferenceError: name is not defined
‚Üí You're using a variable that doesn't exist

Uncaught TypeError: Cannot read property 'length' of undefined
‚Üí You're trying to use .length on something that doesn't exist

Common Debugging Patterns

// 1. Check if variable exists console.log("Is name defined?", name); // 2. Check type of variable console.log("Type:", typeof age); // 3. Check array contents console.log("Array:", myArray); console.log("Length:", myArray.length); // 4. Check if condition is true console.log("Is x > 10?", x > 10); // 5. Mark checkpoints in your code console.log("=== Checkpoint 1 ==="); // ... some code ... console.log("=== Checkpoint 2 ===");

Debugging Workflow

  1. Read the error message
  2. Find the line number
  3. Add console.log() before and after that line
  4. Check what values variables have
  5. Compare to what you expected
  6. Fix the code
  7. Test again!
üí° Debugging Tips:
‚Ä¢ Use descriptive console.log() messages: console.log("User age:", age)
‚Ä¢ Check one thing at a time
‚Ä¢ Take breaks if you're stuck - fresh eyes help!
‚Ä¢ Google error messages - you're not the first person with this error!
‚Ä¢ Ask for help - explain the problem to someone (even a rubber duck!)

Let's Build a Calculator!

Time to put everything together! You'll build a working calculator that can add, subtract, multiply, and divide.

1 Create the HTML Structure
<!DOCTYPE html> <html> <head> <title>Simple Calculator</title> <style> /* Add CSS here */ </style> </head> <body> <div class="calculator"> <h1>Calculator</h1> <input type="number" id="num1" placeholder="First number"> <input type="number" id="num2" placeholder="Second number"> <div class="buttons"> <button onclick="add()">+</button> <button onclick="subtract()">-</button> <button onclick="multiply()">√ó</button> <button onclick="divide()">√∑</button> </div> <div id="result">Result: </div> </div> <script> // JavaScript will go here </script> </body> </html>
2 Add CSS Styling
.calculator { max-width: 400px; margin: 50px auto; padding: 30px; background: white; border-radius: 10px; box-shadow: 0 5px 20px rgba(0,0,0,0.1); } input { width: 100%; padding: 15px; margin: 10px 0; font-size: 18px; border: 2px solid #ddd; border-radius: 5px; } .buttons { display: grid; grid-template-columns: repeat(4, 1fr); gap: 10px; margin: 20px 0; } button { padding: 20px; font-size: 24px; border: none; background: #ff6b6b; color: white; border-radius: 5px; cursor: pointer; transition: 0.3s; } button:hover { background: #ee5a6f; transform: translateY(-2px); } #result { margin-top: 20px; padding: 20px; background: #f8f9fa; border-radius: 5px; font-size: 24px; text-align: center; font-weight: bold; }
3 Write the JavaScript Functions
// Get input values function getNumbers() { let num1 = parseFloat(document.getElementById('num1').value); let num2 = parseFloat(document.getElementById('num2').value); return { num1, num2 }; } // Display result function showResult(result) { document.getElementById('result').textContent = 'Result: ' + result; } // Calculator functions function add() { let { num1, num2 } = getNumbers(); let result = num1 + num2; showResult(result); console.log(`${num1} + ${num2} = ${result}`); } function subtract() { let { num1, num2 } = getNumbers(); let result = num1 - num2; showResult(result); console.log(`${num1} - ${num2} = ${result}`); } function multiply() { let { num1, num2 } = getNumbers(); let result = num1 * num2; showResult(result); console.log(`${num1} √ó ${num2} = ${result}`); } function divide() { let { num1, num2 } = getNumbers(); if (num2 === 0) { showResult("Cannot divide by zero!"); return; } let result = num1 / num2; showResult(result); console.log(`${num1} √∑ ${num2} = ${result}`); }

2. User clicks a button (+, -, √ó, √∑)
3. JavaScript gets the values from the inputs
4. Does the math operation
5. Shows the result on the page
6. Logs the calculation to the console
